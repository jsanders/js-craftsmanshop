Time Dot Com instructions

Use Jasmine to test-drive a simple little web application that uses AJAX to query a remote web service http://time-service.heroku.com and in turn displays the time on the page. The specs should be able to run without Internet access, though.

date.js has been included to make parsing whatever date the server returns easier (example: Date.parse("2011-09-07 17:03:38 -0700")).
